What Is This?

Guitar Mode Finder is a tool for guitarists who want to understand modes, scales, and the CAGED system — without wading through bloated music theory sites, paywalls, or ads.

Pick a key and a mode. The app shows you the notes, the interval formula, the diatonic 7th chords, the pentatonic subset, and a full fretboard diagram — across all five CAGED positions if you want. That's it.

Features

  • Interactive mode selector — choose any of the 7 diatonic modes in all 12 keys
  • Live fretboard diagram — rendered in real time, shows root, pentatonic, and scale tones
  • Diatonic 7th chords — the full chord family for any mode, displayed as a chord readout
  • Pentatonic subset — see which notes form the pentatonic within the mode
  • Scale family browser — shows all 7 modes that share the same parent key
  • CAGED view — toggle to see the mode across all five fretboard positions
  • Day / Night theme — works in any lighting condition
  • No backend, no account — runs entirely in the browser, zero tracking

About the Project

This tool was built out of personal frustration with existing mode references — either too abstract, too cluttered, or locked behind subscriptions. The goal was a fast, focused tool that a guitarist could pull up mid-practice and immediately use.

It's a side project. There are no analytics, no ad networks, no email lists. If something is broken or missing, the feedback page is the right place to say so.

Found a bug? Have a feature idea? Something confusing? Leave feedback — it's read and taken seriously.
